# LinkedIn Daily Posts — 2 Posts/Day

Generate 2 LinkedIn posts daily (News + rotating type: Tips, Carousel, Infographic, Motivation) and track them in Excel. **Local storage only.** No cloud upload.

---

## ? Quick Setup (3 minutes)

### 1. Install Dependencies

\\\ash
pip install openpyxl requests python-dotenv
\\\

### 2. Get API Key

1. Go to: https://openrouter.ai
2. Sign up ? Settings ? API Keys ? Create key
3. Copy the key (looks like: \sk-or-v1-...\)

### 3. Create \.env\ File

\\\ash
# Copy the template
copy .env.example .env

# Edit .env and add your key:
# OPENROUTER_API_KEY=sk-or-v1-YOUR_KEY_HERE
\\\

### 4. Run

\\\ash
python3 fetch_ai_news_rss.py
python3 generate_posts_2post_model.py
python3 update_post_history.py
\\\

Done! 2 posts generated + Excel updated ?

---

## ?? What You Get

- **2 posts/day**: News + rotating type (Tips, Carousel, Infographic, Motivation)
- **Never repeats**: Same type never appears consecutively
- **Excel history**: All posts tracked with metadata (date, text, word count, source)
- **Local storage**: Everything stays on your computer
- **Cost**: ~\.01/day (~30˘/month)

## ?? Output Files

Each day creates:

- \linkedin_posts_YYYYMMDD.txt\ — 2 posts (News + rotating type)
- \post_history.xlsx\ — Excel file (appends 2 rows/day)
- \post-rotation-log.json\ — Tracks which type was posted (prevents repeats)
- Optional: \linkedin-infographic-YYYYMMDD.png\ or carousel PDF (on relevant days)
